Spread

A distraction-free RSVP speed reader for Android.

RSVP (Rapid Serial Visual Presentation) displays one word at a time in a fixed position, eliminating eye movement so you can read faster with better focus.

Features

  • Words anchored at the Optimal Recognition Point (ORP) for stable eye position
  • Long words split at meaningful boundaries (prefixes/suffixes)
  • Adaptive timing: short words flash quickly, long words get more time
  • Zen mode: UI fades away during reading
  • 100-1000 WPM with customizable timing presets
  • Works offline, no accounts, no tracking

For more information, see the GitHub repository.
